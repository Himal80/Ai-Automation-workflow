## 📌 About Project
This is a real AI automation system built using n8n.

It can:
- Auto reply to users (Telegram / Gmail)
- Process user requests using AI
- Save data automatically in Google Sheets

---

## ⚙️ Workflow Process

1. User sends message  
2. AI Agent processes input  
3. Data saved in Google Sheet  
4. Auto reply sent to user  

---

## 🛠️ Tools Used
- n8n
- Telegram Bot / Gmail
- Google Sheets
- AI Chat Model

---

## 🚀 Use Case

This system helps businesses:
- Automate customer support  
- Save time  
- Manage data automatically
